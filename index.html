AI Travel Planner

Welcome to AI Travel Planner! This application helps you create personalized travel itineraries using Google's Gemini AI.

Features:

  • 🌍 Personalized travel recommendations
  • 📅 Day-by-day itinerary planning
  • 💰 Smart budget calculation and tracking
  • 🏨 Accommodation suggestions
  • 🍴 Local food recommendations
  • 🎯 Must-see attractions with Google Maps integration
  • 📥 Downloadable travel plans

To get started with the AI Travel Planner, please follow the setup instructions in our GitHub repository.

Note: This is a self-hosted application. You'll need to set up your own instance with your Gemini API key.
